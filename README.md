# 📊 FX Rate Trend & Business Impact Analysis

## Overview

Analyzed the impact of foreign exchange (FX) rate fluctuations on international client transactions during **July 2023** using Microsoft Excel. The project evaluates how currency movements influenced transaction values and supports data-driven financial planning.

> **Portfolio Case Study:** Developed using a fictional company with realistic business data.

## Business Objective

Assess the financial impact of exchange-rate movements on international transactions and present actionable business insights.
